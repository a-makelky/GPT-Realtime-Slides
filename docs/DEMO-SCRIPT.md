# Demo video script

Target runtime: 2:10

## 0:00 to 0:20: The problem

"Presentation software assumes the presenter is standing beside a keyboard. GPT Realtime Slides keeps the human with the room while keeping every AI action deterministic."

Show the opening slide and advance with the keyboard.

## 0:20 to 0:50: One source, two views

Open the presenter and audience routes side by side.

"Both views read the same deck source. The audience gets only the story. Notes, controls, microphone state, and the command sandbox exist only in the presenter view."

Open Notes, then show they are absent from the audience page.

## 0:50 to 1:25: Live audience input

Open the audience-data slide, grouped results, and the entrance form.

"Cloudflare D1 pairs anonymous entrance and exit ratings. The production database started empty. Public results appear only after three paired responses."

## 1:25 to 1:45: Working no-key sandbox and Realtime design

Run `show audience results`, then show the voice slide.

"The hosted demo needs no OpenAI API key. Keyboard, buttons, commands, and optional voice use the same deterministic controller."

## 1:45 to 2:00: Codex and GPT-5.6

Show the commit history, privacy test, and README build narrative.

"Codex and GPT-5.6 helped turn a working event tool into a reusable public product. Parallel subagents audited privacy, licensing, Git history, Cloudflare architecture, and the extraction boundary."

## 2:00 to 2:10: Close

Return to the closing slide and show the QR.

"Fork it, change one deck file and six theme tokens, and make it yours. GPT Realtime Slides is open source under the MIT license."
